# C-Sharp_WindowSize_Changer
## 概要
* ウインドウサイズを変更できるアプリです。

## 使い方
① 解像度を変えたいアプリをクリックしてください。<br>
② +ボタンと-ボタンで調整してください。<br>
③ 適用ボタンで変えることができます。<br>
